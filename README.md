# Valorant Match Outcome Prediction

This project uses machine learning to predict professional Valorant match outcomes based on player performance statistics.

## Tools Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

## Project Contents
- Data cleaning and preprocessing
- Random Forest classification
- Model evaluation with confusion matrix and classification report
- K-Means clustering for player performance profiles
- PCA visualization

## Main Result
The Random Forest model achieved about 70% accuracy, showing that player performance metrics have useful predictive power, while still leaving room for team-level and contextual factors.
