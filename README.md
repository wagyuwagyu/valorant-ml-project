# Valorant Match Outcome Prediction

This project uses machine learning to predict professional Valorant match outcomes based on player performance statistics. The main notebook covers data cleaning, preprocessing, classification, clustering, and visualization.

## Tools Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Jupyter Notebook

## Project Contents

- Data cleaning and preprocessing
- Random Forest classification
- Model evaluation using confusion matrix and classification report
- K-Means clustering for player performance profiles
- PCA visualization

## Main Result

The Random Forest model achieved about 70% accuracy, showing that player performance metrics have useful predictive power while still leaving room for team-level and contextual factors.
